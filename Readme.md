## Business Problem

An e-commerce company has launched a new recommendation engine that suggests products to users while they browse.

The company wants to know:

  "Does the new recommendation engine actually improve customer purchases, or is the observed increase just due to random chance?"

As the data scientist, your job is to analyze the experiment and recommend whether the company should roll out the new recommendation engine.

## Dataset Overview

Each row represents one customer session on the website.

The dataset is synthetically generated, but it mimics how an actual e-commerce company might log user behavior.

## Columns
- Customer Information
- user_id → Unique identifier for each customer.
- age → Customer's age.
- gender → Male/Female/Other.
- country → Customer's country.
- device → Mobile, Desktop, or Tablet.
